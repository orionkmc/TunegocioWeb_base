TunegocioWeb_crm conecta con los usuarios de wordpress y brindar a los administradores del sitio web un mejor seguimiento de sus contactos.

== Description ==

Este CRM tiene el objetivo principal de conectar con los usuarios de wordpress y brindar a los administradores del sitio web una mejor expriencia de seguimiento y atencion a los contactos. 
Tenemos como visión ir desarrollando Addons que conecten con este CRM Base y asi ir ampliando las funcionalidades del mismo. 
Para los conocedores de la filosofia CRM nuestra mision es brindar una herramienta que permita aumentar las ventas de cualquier empresa y un mejor seguimiento a contactos, clientes, prospectos y mas.
Para el envio de cualquier sugerencia por favor escribir a info@tunegocioweb.info ya que sabemos que aun estamos en vias de desarrollo y necesitaremos muchos de sus opiniones para creacer y ser un gram plugins para ustedes.
Thanks por usar nuestro plugins

== Installation ==

1.- Cargar los archivos del plugin para el directorio '/wp-content/plugins/' , o instalar el plugin a través de la pantalla de plugins de WordPress directamente.

2.- Activar el plugin a través de la pantalla de plugins en WordPress
